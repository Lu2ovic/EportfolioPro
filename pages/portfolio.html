Portfolio de compétence

Contexte :

Dans le cadre de ma première année en BUT Métier du Multimédia et de l’Internet, j’ai dû réaliser un site portfolio de compétences.
Le but du site est d’avoir des traces des différents projets réalisés lors du BUT.
Pour chaque projet, il y a une description avec des informations supplémentaires, et une explication de comment on a réalisé le projet.

Réalisation :

Pour créer mon site, j’ai d’abord réfléchi aux pages qu’il me fallait, au style, puis aux données utiles pour le site. J’ai commencé en réalisant une maquette de mon site via Figma.
Pour coder le site, j’ai utilisé l'éditeur de code Visual Studio Code. Le site a été codé en HTML, CSS, JavaScript et en PHP. J’ai réalisé le filtre de projet avec du CSS.
Pour avoir une meilleure expérience utilisateur, j’ai utilisé du CSS et du JavaScript pour ajouter des animations lorsque la souris passe sur un élément cliquable, ou encore pour donner des informations supplémentaires.
Le site doit être généré grâce à une base de données.
J’ai donc réalisé un MCD (modèle conceptuel de données), puis un MLD (modèle logique de données). J’ai ensuite conçu la base de données via PHPmyAdmin. Il m’a donc fallu implémenter du PHP dans mes pages html pour pouvoir introduire les données récupérées dans la base de données.
Pour ce faire, j’ai utilisé la méthode MVC (Modèle Vue Contrôleur). Dans la page “Modèle”, j’ai réalisé la connexion à la base de données, mais on y retrouve aussi les requêtes SQL permettant de récupérer les données.
